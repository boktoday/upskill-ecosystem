🧠 AI Agent Flashcards

What is an AI Agent Framework?

The scaffolding that allows an LLM to perform tasks beyond generating text — providing tools, memory, and action capabilities.

What are the 5 Core Capabilities?

1. Autonomous Execution
2. Tool Use
3. Memory & Context
4. Decision Making
5. Action Chaining

API vs Webhook?

API: You ask, it answers
Webhook: It notifies you when something happens (doorbell)
