💡 How Ollama Works

Ollama downloads AI models to your computer and runs them locally. It exposes an OpenAI-compatible API on port 11434 — meaning OpenClaw connects to it just like any other API provider. Your computer becomes an AI server.

Install Ollama (One Line)

# Linux / Mac / WSL curl -fsSL https://ollama.com/install.sh | sh

Pull Qwen3.5 9B

# Download and run Qwen3.5 9B (one time) ollama pull qwen3.5:9b # Verify it's installed ollama list

Test the API (OpenAI-compatible)

# Quick test — OpenAI format curl http://localhost:11434/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model": "qwen3.5:9b", "messages": [{"role": "user", "content": "Hello!"}]}'

OpenClaw Config

{ "models": [ { "provider": "openai", "name": "qwen3.5:9b", "apiKey": "ollama", // no real key needed — local "baseUrl": "http://localhost:11434/v1" } ] }

💻 Hardware needed: Qwen3.5 9B needs ~6GB RAM above your OS. Works great on any modern laptop with 16GB RAM. Use ollama run qwen3.5:9b to start chatting directly.
