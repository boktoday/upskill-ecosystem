LLM (Large Language Model)

AI model trained on vast text data, capable of understanding and generating human-like text.

API (Application Programming Interface)

A set of rules allowing software applications to communicate. You ask, it answers.

Webhook

HTTP callback that notifies you when something happens. Like a doorbell - alerts you to events.

Function Calling

Ability for AI to call custom functions/APIs during generation to perform actions.

Prompt Engineering

The practice of designing effective inputs to get desired outputs from LLMs.

Context Window

The amount of text an LLM can consider at once when generating responses.

Temperature

Controls randomness in LLM output. Low = focused/deterministic, High = creative/varied.

Token

Basic unit of text a model reads/writes. ~4 characters = 1 token.
