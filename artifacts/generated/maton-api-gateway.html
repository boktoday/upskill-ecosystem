1 What Is the Maton API Gateway?

The Maton API Gateway is an OpenClaw skill that lets your AI agent connect to 100+ third-party APIs with managed OAuth. Instead of building OAuth flows for every service, Maton handles it all — you just make simple API calls.

Instead of: OAuth app registration → Auth flow → Token refresh → API calls

You do: Set API key → Make API calls directly

2 Supported Services

Google Workspace Microsoft 365 GitHub Slack Notion Airtable HubSpot Salesforce Jira Linear Stripe Shopify Gmail API Google Calendar Google Drive Outlook Discord + 80 more

3 Setup in 3 Steps

Step 1 — Install the Skill

openclaw skills install api-gateway

Step 2 — Get Your API Key

  1. Sign up at maton.ai
  2. Go to Settings → API Key
  3. Copy your API key

Step 3 — Set Environment Variable

export MATON_API_KEY="your_matон_api_key_here"

4 The Base URL Pattern

All API calls use this format:

https://gateway.maton.ai/{app}/{native-api-path}

Replace {app} with the service name and {native-api-path} with the actual API endpoint.

Important: The URL path MUST start with the app name (e.g. /google-mail/...). This tells Maton which OAuth connection to use. For Gmail, the native path starts with gmail/v1/ — full path: /google-mail/gmail/v1/users/me/messages

5 Authentication Header

Every request needs this header — Maton injects the OAuth token automatically:

Authorization: Bearer $MATON_API_KEY

6 Quick Start Examples

Send a Slack message

import urllib.request, os, json data = json.dumps({'channel': 'C0123456', 'text': 'Hello from gateway!'}).encode() req = urllib.request.Request( 'https://gateway.maton.ai/slack/api/chat.postMessage', data=data, method='POST' ) req.add_header('Authorization', f'Bearer {os.environ["MATON_API_KEY"]}') req.add_header('Content-Type', 'application/json') print(json.dumps(json.load(urllib.request.urlopen(req)), indent=2))

List Slack connections

import urllib.request, os, json req = urllib.request.Request('https://ctrl.maton.ai/connections?app=slack&status=ACTIVE') req.add_header('Authorization', f'Bearer {os.environ["MATON_API_KEY"]}') print(json.dumps(json.load(urllib.request.urlopen(req)), indent=2))

Send a Slack message

const res = await fetch('https://gateway.maton.ai/slack/api/chat.postMessage', { method: 'POST', headers: { 'Authorization': `Bearer ${process.env.MATON_API_KEY}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ channel: 'C0123456', text: 'Hello!' }) }); const data = await res.json(); console.log(JSON.stringify(data, null, 2));

List connections

const res = await fetch('https://ctrl.maton.ai/connections?app=slack&status=ACTIVE', { headers: { 'Authorization': `Bearer ${process.env.MATON_API_KEY}` } }); const { connections } = await res.json(); console.log(connections);

Send a Slack message

curl -X POST https://gateway.maton.ai/slack/api/chat.postMessage \ -H "Authorization: Bearer $MATON_API_KEY" \ -H "Content-Type: application/json" \ -d '{"channel": "C0123456", "text": "Hello from gateway!"}'

List connections

curl "https://ctrl.maton.ai/connections?app=slack&status=ACTIVE" \ -H "Authorization: Bearer $MATON_API_KEY"

7 Connection Management

Manage OAuth connections via ctrl.maton.ai — separate from the gateway API.

List connections

GET https://ctrl.maton.ai/connections?app=slack&status=ACTIVE

Query params: app (service name), status (ACTIVE | PENDING | FAILED)

Create a connection

POST https://ctrl.maton.ai/connections Body: {"app": "slack", "method": "OAUTH2"} → Returns a URL — open it in a browser to complete OAuth

Get connection status

GET https://ctrl.maton.ai/connections/{connection_id}

Delete a connection

DELETE https://ctrl.maton.ai/connections/{connection_id}

8 Using Inside OpenClaw

Once installed and configured, use the skill inside any OpenClaw agent. The agent can call APIs directly via exec or a dedicated tool — letting it:

  • Send Slack messages on your behalf
  • Read and write to Google Sheets
  • Create GitHub issues and PRs
  • Query HubSpot CRM data
  • Read/write Notion databases
  • Pull data from Airtable
💡

Your OpenClaw agent becomes a universal API client. Tell it in plain English what to do — "send a message to #sales on Slack" — and it constructs and executes the API call automatically.
