# Students Data

This folder stores individual student data.

## Structure

Each student gets a unique ID folder:
```
students/
├── student-001/
│   └── USER.md
├── student-002/
│   └── USER.md
└── ...
```

## Creating a New Student

When a new student starts:
1. Create new folder: `student-[next-number]/`
2. Copy template into USER.md
3. Fill in discovery answers

## What We Track

- Discovery answers (8 questions)
- Module progress
- Quiz scores
- Spaced repetition schedule
- Notes
